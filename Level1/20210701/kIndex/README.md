# k번쨰 수

## 원하는 것

array의 i번째 숫자부터 j번째 숫자까지 자르고 정렬했을 때, k번째에 있는 수

## 아는 것

찍는 방식

1. 1번: 1,2,3,4,5
2. 2번: 2,1,2,3,2,4,2,5
3. 3번: 3,3,1,1,2,2,4,4,5,5

- answers: 정답 배열

## 조건

- answers.length >= 10,000
- 문제의 정답은 1, 2, 3, 4, 5중 하나
- 가장 높은 점수를 받은 사람이 여럿일 경우, 오름차순 정렬

## 계획

- 문제의 개수만큼 패턴을 만든다.
- 문제의 답과 비교한다.
- 가장 많이 맞춘 번호를 얻는다.
