# 최대공약수와 최소공배수

## 원하는 것

[최대공약수, 최소공배수]

## 아는 것

1. 입력받은 두 자연수 n, m
2. 최대공약수 : 두 수를 나누어 떨어지는 수 중 가장 큰 수
3. 최소공배수 : 두 수로 나누어 떨어지는 수 중 가장 작은 수

## 조건

1. 1 <= n, m <= 1000000

## 계획

1. 최대공약수를 구한다.
2. 최소공배수를 구한다.

## 반성

- 배열로 만들어 reduce로 순회하는 것보다 for문으로 거꾸로 순회하다가 값을 구할 때 순회를 멈추는 것이 훨씬 순회를 적게 한다.
- 풀고나니 유클리드 호제법이라는 공식이 있었다. 공식을 외우자.
