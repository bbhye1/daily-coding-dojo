# 문제1
## 최소힙

### 원하는 것
0이 나올 때마다 최소힙을 꺼내서 배열로 모아 반환하기 

### 내가 아는 것
- 입력값: 입력되는 숫자 배열 numbers
### 조건
1. 자연수가 입력되면 최소힙에 입력한다.
2. 숫자 0 이 입력되면 최소힙에서 최솟값을 꺼내어 출력한다. (출력할 자료가 없으면 -1를 출력한다.)
3. -1이 입력되면 프로그램 종료한다.

### 계획
이진트리 최소힙을 구현해보자.
