# 부족한 금액 계산하기

## 원하는 것

놀이기구를 count번 이용할 때 자신의 금액에서 얼마가 모자라는지 

## 아는 것

- 원래 이용료: price
- 이용할 횟수: count
- 현재 금액: money

- N번째 이용하면 이용료의 N배를 받는다.

## 조건

1. 1 ≤ price ≤ 2,500, price는 자연수
2. 1 ≤ money ≤ 1,000,000,000, money는 자연수
3. 1 ≤ count ≤ 2,500, count는 자연수

## 계획

답 = 모두 연산한 후 이용할 횟수일 때 금액 - 현재 금액
