# 조이스틱

## 원하는 것

조이스틱 조작 횟수의 최솟값

## 아는 것

- A로만 이루어진 문자열로 시작
- 위 : 다음 알파벳
- 아래: 이전 알파벳
- 왼: 커서를 왼쪽으로 이동
- 오른: 커서를 오른쪽으로 이동
- 만들고자 하는 이름: name

## 조건

- name은 알파벳 대문자로만 이루어짐
- 1 <= name의 길이 <= 20

## 계획

글자를 바꾸는데 드는 횟수의 최솟값: 아래 오른쪽 이동 중 적은 횟수의 이동

위치를 변경하는 데 드는 최솟값: 지금 현재에서 왼쪽 오른쪽으로 이동했을때에 적은 횟수의 이동
